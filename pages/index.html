    txtprint

    a project by Jeremy Jacob for Full Steam Ahead

    AE Wright Middle School, Robotics 8th Grade

    txtprint is a largely 3d printed portable thermal printer designed for students and classroom use. It uses the Chrome WebUSB library to communicate with an Arduino-style board. I made it because from personal expirience, being able to quickly print conclusions or other text in class is an invaluable asset. I was a group member in a previous FSA project, Classmate. That project also used an Arduino, so I have expirience with the boards and C. As for web development, it is largely new to me. This is the first full website I have built enitely from scratch. Feel free to take a peek at the source code for both the website and Arduino sketch on the github page. If you have feedback for the project or website, see the form on the FAQ page, or email me at
